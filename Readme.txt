Movie Recommendation System


main.lp is the file which we will going to find answerset. It contains rules for mapping the 
person's preferences with movie's categories.

movies.lp contains movie data with movie name, its category and region.

category.lp is file which contains facts about the categories of movies.

hobbies.lp is the file containing rules about person's hobbies and its mapping to the genres.


movies.lp , hobbies.lp & category.lp, these files are included into main.lp file.
